## ✨ Core Features

Our platform boasts a dedicated **Role-Based Architecture** allowing for a customized experience based on user type:

### 🧑‍💼 For Job Seekers
* **Seamless Search:** Filter through live databases of job openings by location, industry, or keywords.
* **Instant Applications:** Upload your custom cover letter and resume link securely.
* **Interactive Dashboard:** Track the live status of all your applications (Pending, Accepted, Rejected).

### 🏢 For Employers
* **Posting Management:** Easily draft, edit, and publish job opportunities to the public board.
* **Candidate Review:** A dedicated applicant tracking system to view resumes and update candidate statuses.

### 🛡️ For Administrators
* **Mock API Integration:** One-click functionality to securely fetch dummy data and test the database capacity.
* **Community Moderation:** Full control to delete fraudulent postings or ban users if necessary.

## 🛠️ Technology Stack
We prioritized a lightweight, lightning-fast architecture for this SaaS platform.

* **Backend Engine:** `Python 3` with `Flask` micro-framework.
* **Security Layer:** `Flask-Login` session management and `Werkzeug` secure password hashing.
* **Relational Database:** `SQLite` coupled with `SQLAlchemy` ORM.
* **Frontend UI/UX:** `Bootstrap 5`, `HTML5/CSS3`, custom glassmorphism styling, and SVGs.
* **Hosting Pipeline:** Containerized and hosted natively on `Render.com` running `Gunicorn`.

## 🚀 Local Installation

*To run this project locally on your machine:*

1. Open your terminal and clone the repository.
   ```bash
   git clone https://github.com/Himasri2706/job-portal.git
   ```

2. Install all strict environmental dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Launch the local web server:
   ```bash
   python app.py
   ```
   *(The relational database will automatically instantiate itself upon first launch.)*

4. Visit `http://127.0.0.1:5000` in your web browser.

> **Note:** A default admin account is automatically configured!  
> **Username:** `admin` | **Password:** `admin123`
